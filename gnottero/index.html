This datapack allows you to bring your
beacons everywhere.To install this datapack,
just drag the zip file into the "Datapack"
folder located in the folder of each world from 1.13.
After that, just enter the game
and run the "/reload" command for safety.
For more information, run the command
"/function portablebeacon:guide" to get
a complete guide about the datapack.
