Scaricare, estrarre ed avviare MultiMC ( download )
Creare un'istanza selezionando la versione 1.16.2, cliccare con il tasto destro sull'istanza creata e successivamente su Modifica istanza.
Aggiungere fabric all'istanza mediante l'apposito bottone posto sulla destra.
Andare nella voce Mods del menù posto sulla sinistra, aprire la cartella mediante l'apposito bottone nell'angolo in basso a destra della finestra e importare in tale cartella le seguenti mod:
  • MaliLib ( download | informazioni )
    [Necessaria per Litematica, Tweakeroo, ItemScroller e MiniHUD]
  • Litematica ( download | informazioni )
  • Tweakeroo ( download | informazioni )
  • ItemScroller ( download | informazioni )
  • MiniHUD ( download | informazioni )
  • CarpetMod ( download | informazioni )
  • WorldEdit ( download | informazioni )
  • VoxelMap ( download | informazioni )
  • Fabric API ( download | informazioni ) [Necessaria per ReplayMod, Logical Zoom e LambDynamicLights]
  • ReplayMod ( download | informazioni )
